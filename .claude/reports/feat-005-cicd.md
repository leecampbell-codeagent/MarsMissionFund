## Potential Concerns

None blocking.

**Informational:** The `unit-and-integration-tests` job installs dbmate via a direct GitHub release download (`https://github.com/amacneil/dbmate/releases/latest/download/dbmate-linux-amd64`). The `latest` tag means the CI always uses the newest dbmate binary, which could introduce unexpected breakage if a dbmate major version changes the migration format. For production stability, this should be pinned to a specific version. This is a pre-existing pipeline concern, not introduced by feat-005.
