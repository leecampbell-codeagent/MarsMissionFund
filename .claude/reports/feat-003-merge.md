# Merge Report: feat-003 — Authentication Integration (Clerk)

> Generated by Pipeline Orchestrator after all quality gates passed.

## Summary

feat-003 delivers full Clerk authentication integration for Mars Mission Fund. The backend uses `@clerk/express` for JWT verification on all protected routes with a lazy user-sync pattern (upserts `users` + `user_roles` on first authenticated request). The frontend uses `@clerk/react` with sign-in/sign-up pages, a protected `/dashboard` route, the `useApiClient` hook with per-request JWT injection, and a header component with sign-in/sign-out. A `MOCK_AUTH=true` mode supports local development and testing without real Clerk credentials.
