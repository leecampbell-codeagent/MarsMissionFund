## Changelog Entry

### feat-003: Campaign Creation, Submission, and Review Pipeline

**Backend:**
- `POST /api/v1/me/roles/creator` — creator role self-service designation (KYC required)
- `POST /api/v1/campaigns` — create campaign draft
- `PATCH /api/v1/campaigns/:id` — auto-save draft fields (lenient validation, G-026)
- `POST /api/v1/campaigns/:id/submit` — submit draft for review (strict validation)
- `GET /api/v1/campaigns/:id` — view campaign (role/status-gated)
- `GET /api/v1/me/campaigns` — list creator's own campaigns
- `GET /api/v1/campaigns/review-queue` — FIFO queue for reviewers
- `POST /api/v1/campaigns/:id/claim` — reviewer claims campaign
- `POST /api/v1/campaigns/:id/approve` — reviewer approves with notes
- `POST /api/v1/campaigns/:id/reject` — reviewer rejects with reason + guidance
- `POST /api/v1/campaigns/:id/launch` — creator launches approved campaign
- `POST /api/v1/campaigns/:id/archive` — creator/admin archives campaign
- `POST /api/v1/campaigns/:id/reassign` — admin reassigns reviewer
- New domain: `Campaign` entity, 23 domain errors, 7-status state machine
- New adapters: `PgCampaignRepository`, `PgCampaignAuditRepository`
- New application service: `CampaignAppService` (9 methods)

**Frontend:**
- `/campaigns/new` — Campaign Create page with multi-section auto-save form
- `/campaigns/:id` — Campaign Detail page with role-gated action panels
- `/me/campaigns` — My Campaigns page with status filters
- `/review-queue` — Review Queue page with FIFO indicator and overdue tags
- 8 new components: CampaignCard, CampaignForm, CampaignStatusBadge, MilestoneList, ReviewActionPanel, RejectionFeedbackPanel, ReviewQueueList, RiskSection/TeamSection/BudgetSection/MilestonesSection
- New hooks: useCampaign, useCampaigns, useCampaignActions
- New API client methods for all 12+ campaign endpoints

**Database:**
- Migration `20260305150000_create_campaigns_table.sql` — campaigns table with JSONB arrays, BIGINT monetary columns, all indexes, FK constraints, ON DELETE RESTRICT for creator
- Migration `20260305151000_create_campaign_audit_events_table.sql` — append-only audit trail
