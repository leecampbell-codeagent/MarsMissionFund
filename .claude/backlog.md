## MVP Scope Boundary

All 15 features above represent the MVP scope as defined in the product vision (L1-001) across three core workflows:

1. **Campaign Lifecycle** (feat-002, feat-005, feat-006, feat-007, feat-008, feat-013): Creator submits → review pipeline → live campaign → milestone settlement → complete
2. **Donor Lifecycle** (feat-009, feat-010, feat-011, feat-012): Discover → contribute → track impact
3. **Account and Identity** (feat-001, feat-002, feat-003, feat-004, feat-005): Register → onboard → verify identity
